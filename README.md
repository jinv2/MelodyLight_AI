## 🌟 项目定位
本智能体是专为**小学一年级音乐常识教育**打造的多模态交互系统。基于“场景即教学”的理念，将中国传统文化瑰宝《山海经》与现代音律算法相结合，旨在通过 AI 技术实现儿童的乐律启蒙与文化溯源。

## 🧠 技术架构：Shensist-Matrix (L-A-T-T 逻辑)
本项目拒绝繁琐的传统网页交互，采用极致精简的 **L-A-T-T** (Leverage, AI Enhance, Trim, Target) 架构：
- **Leverage (杠杆)**：利用 Web Audio API 物理建模算法仿真传统乐音。
- **AI Enhance (AI 增强)**：基于神经网络生成的 16:9 沉浸式山海经世界观。
- **Trim (修剪)**：零文字障碍界面，完全适配一年级儿童认知水平。
- **Target (目标)**：建立色彩、音高、神兽三位一体的记忆映射。

## 🐉 七律神兽系统
智能体内置七位乐律守护神，每位神兽对应特定频率与色彩：
1. **白泽 (1 - C4)**：炽热红 - 智慧之音
2. **麒麟 (2 - D4)**：活力橙 - 仁兽之律
3. **貔貅 (3 - E4)**：璀璨金 - 招福之声
4. **凤凰 (4 - F4)**：翡翠绿 - 祥瑞之鸣
5. **应龙 (5 - G4)**：苍穹蓝 - 深远之响
6. **甪端 (6 - A4)**：幽邃蓝 - 光明之奏
7. **青鸾 (7 - B4)**：梦幻紫 - 优美之旋

## 🚀 快速启动 (面向学校部署)
### 1. 环境准备
确保已安装 Python 3.10+ 环境。
```bash
pip install fastapi uvicorn
```

### 2. 运行智能体
```bash
python main.py
```
访问 `http://localhost:8000` 即可进入山海经大乐园。
