# MelodyLight AI: 山海寻宝大乐园

基于 Shensist-Matrix 架构开发的艺术智能体，专为一年级美育设计。通过 7 位山海经神兽（白泽、麒麟等）引导孩子进行乐律探索。

## 🚀 快速启动
1. 安装依赖：`pip install -r requirements.txt`
2. 启动智能体：`python main.py`
3. 访问地址：`http://localhost:8000/interface/`

## 🎨 核心特性
- **L-A-T-T 逻辑**：极致精简的交互界面，符合“琴棋书画”寻宝大比拼标准。
- **七律神韵**：音符、神兽、色彩三位一体映射，支持 1-3-5 和弦共鸣。
- **物理建模音效**：基于 Web Audio 的中国传统乐律仿真（Triangle Wave ADSR 引擎）。
- **成就系统**：实时节奏 Combo 追踪与神兽灵气反馈。
